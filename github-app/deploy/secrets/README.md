# Private deployment secrets

Create these files locally on the deployment host. Never commit their contents.

- `github-app-private-key.pem` — the PEM private key generated by the private GitHub App registration.
- `github-webhook-secret.txt` — the exact webhook secret configured in the GitHub App registration.

Recommended permissions:

```bash
chmod 700 secrets
chmod 600 secrets/github-app-private-key.pem secrets/github-webhook-secret.txt
```

The Docker Compose stack mounts both files read-only under `/run/secrets/`.
